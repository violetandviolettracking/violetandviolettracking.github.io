Violet and Violet Tracking

Android Application

About

Android application interfaced with portable Bluetooth printer and card reader, a distributed client with centralised server to spot challan traffic violators and to keep track of vehicle theft with RTO database integrated. Violet tracking application is using to track particular user or all user location in map view.

  • Client
    Violet and Violet Tracking is implemented for Chennai Traffic Police.
  • Printer
    Violet application is communicate with internal and external printer.
  • Card Reader
    Credit card and debit card reader is available in Violet application.
  • communicate
    Violet application is communicate with Rest API and websocket.
